#  virtuelle Maschine mit Vagrant
Die virtuelle Maschine ist in _*Vagrantfile*_ beschrieben.

- erstellen einer VM
```
$ vagrant up
```
- login in die VM
```
$ vagrant ssh
```
- info zu libvirt-Provider
  - https://github.com/vagrant-libvirt/vagrant-libvirt.git
